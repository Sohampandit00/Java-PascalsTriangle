# Java-PascalsTriangle

A Java program that generates and prints Pascal's Triangle based on the number of rows specified by the user.

## Features

* Prompts the user to enter the number of rows to generate.
* Generates and prints Pascal's Triangle with the specified number of rows.
* Handles user input and displays the triangle in a formatted manner.

## Technologies Used

* Java

## How to Run

1.  Ensure you have the Java Development Kit (JDK) installed.
2.  Compile the `PascalsTriangle.java` file using `javac PascalsTriangle.java`.
3.  Run the compiled program using `java PascalsTriangle`.
4.  Follow the on-screen prompts to enter the number of rows.
